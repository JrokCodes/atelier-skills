# Stitch Design Reference Sites

> Curated collection of premium websites and SaaS products to reference in Stitch prompts.
> When starting a new Stitch project, pick 2-3 references that match your target aesthetic.

---

## How to Use References in Stitch Prompts

Reference sites help Stitch understand the QUALITY LEVEL and AESTHETIC you want. Include them as:

```
DESIGN DIRECTION: Premium dark-mode SaaS quality. Reference aesthetic: Linear.app for layout,
Vercel dashboard for data cards, Stripe for table design.
```

Don't reference more than 3 sites per prompt — it confuses the output. Pick the most relevant one for each design element.

## Dark Mode SaaS Dashboards

### Linear.app
- **URL**: linear.app
- **Why it's reference-worthy**: The gold standard for dark SaaS UI. Masterful use of subtle borders, glass morphism, whitespace, and typography hierarchy.
- **What to reference**: Sidebar navigation, issue boards, kanban views, overall dark aesthetic, command palette, keyboard-first design philosophy
- **Design tokens**: Near-black backgrounds, very subtle borders, accent colors for interactive states, system font stack with strong weight hierarchy
- **Best for**: Any dark dashboard, project management tools, developer tools
- **In a prompt**: "Design in the style of Linear.app — clean dark mode, subtle borders, generous whitespace, strong typographic hierarchy"

### Vercel Dashboard
- **URL**: vercel.com/dashboard
- **Why it's reference-worthy**: Sleek deployment dashboard with excellent data presentation. Perfect balance of data density and readability.
- **What to reference**: Deployment status cards, analytics charts, project cards, clean tables, domain management UI
- **Design tokens**: Pure black (#000) background, white text, minimal accent colors, Geist font
- **Best for**: Analytics dashboards, deployment tools, DevOps portals, status monitoring
- **In a prompt**: "Reference Vercel's dashboard for data card design — minimal, high-contrast, clean metrics"

### Raycast
- **URL**: raycast.com
- **Why it's reference-worthy**: Premium dark UI with the best glass morphism implementation in production. Beautiful input fields and search patterns.
- **What to reference**: Command palette, glass morphism cards, search/input fields, extension cards, settings UI
- **Design tokens**: Dark backgrounds with glass morphism overlays, vibrant accent colors, rounded corners, subtle gradients
- **Best for**: Search-heavy interfaces, command palettes, tool launchers, glass morphism designs
- **In a prompt**: "Reference Raycast for glass morphism card styling — semi-transparent backgrounds with backdrop-blur"

### Railway.app
- **URL**: railway.app
- **Why it's reference-worthy**: Modern infrastructure dashboard with excellent dark mode. Great project cards and deployment flows.

### Supabase Dashboard
